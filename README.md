# Homebrew Tap for Alpha CLI

CLI tool for cross-referencing congressional trades with prediction markets.

## Installation

```bash
brew tap austron24/alpha-cli
brew install alpha-cli
```

## Usage

```bash
# View congressional trades
alpha congress trades

# View Kalshi markets
alpha kalshi markets

# Cross-reference trades with markets
alpha scan
```
